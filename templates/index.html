PDF Receipt Data Extractor

August 26, 2021

As far as I know, Ontario Health generates two kinds of PDF receipts:

  1. One generated at vaccination time, that's emailed. This is a simple PDF file.
  2. One that can be generated and downloaded by visiting the Ontario COVID-19 vaccination service website. This version is a 'duplicate' with a watermark. This is also digitally signed using the Government of Ontario's PKI but it's impossible to validate the signature outside of the Government of Ontario.

This tool simply takes a PDF file, and returns JSON with some data extracted from it, if it appears to be a valid receipt from Ontario Health. Please keep in mind that this is crude "reverse engineering" based on a small sample of valid receipts and that receipts can be forged.

Extract data from a receipt

PDF files uploaded here are only stored in memory of the server long enough to extract certain data elements from the contents. Data extracted from PDF files uploaded here are not retained. Note: The maximum file size is 256K.
